Retrospective - Start, Stop & Watch For

 

This retro game for holidays we want to reflect specifically on the past 2 sprints ONLY. How do we execute this game?

  1. Start on white board or post it.
  2. Discuss Categories of items - We need to do (Start), We need to stop doing (STOP) and Things we need to remain vigilent for (WATCH For)
  3. List items on category
  4. Discuss and brianstorm

 

START 

  1. Start Implementing / Socializing the PoC's standards performed from CTP to the Application teams - This is of utmost importance - take your work to the Leads

      2. Start Reviewing the Application documentation on TC Wiki as much to familiarize the behavior of any given application  - Socialize and pair relation with App DEV Team/ App Leads to learn as much possible

      3. Key delivery focus for the immediate goals around the AWS Infra Pipeline for Hosting Apps & parallely increment towards integrating tools in increment for the pipeline

      4. Rehearse and come prepared to the Demo and present the Demo to the context of the story - Welcome feedback and any increment from the PO & Team

      5. Be prompt to the stand up - We agreed to a time ; We stick to the time - 

STOP 

  1. Attending Discussion with NO Agenda's or where there is NOT a Context set (Internal/ External)

      2. Stop being Late to any of the scrum events (It's our event  - This is self implied)

      3. Stop presenting progress on work items that are not in one's own current Sprint priority 

      4. Stop finishing work items at the last day 2 hours before the Sprint Ends - This is an Utter BAD practice

WATCH FOR 

  1. Adoption and socialization of standards from CTP back to App teams - All we do in a return on value for the App teams - To better or to simplify what they need
  2. Deviation and context switching losing focus at any situation attend the priority on hand first for Sprint prior to committing any effort externally 

      3. Measure and ensure desk check as much as possible with the PO and ask any team member for inputs/ HELP as one needs


Actions

  • Team attendance for 9:30 AM CT for stand up and Clear focus on the current individual sprint work item priority (TEAM)
  • Increment all efforts towards the goal of Platform Infrastructure which is the current Milestone (TEAM)
