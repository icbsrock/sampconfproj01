Browser Extension (Firefox & Chrome)

This is for the latest browser plugin leveraging new browser web API for Google Chrome & Mozilla Firefox.  Unfortunately, the plugin has not been signed for the respective web-app-stores.  It should be officially released sometime in May, but for now you can install it as a “temporary add-on”. 

(This streamlines the process of recording a test script)

Download here (Firefox or Chrome):

  • http://files.apicasystem.com/Recording+Extension.zip

 

Installation steps for Chrome :

·         Unzip ”Recording Extension.zip” file

·         Open Chrome

·         Navigate to chrome://extensions in the URL bar

·         Click "Load unpacked extension"

·         Select the extension's directory

The extension will be installed, and will stay installed until you disable or remove it from Chrome.
