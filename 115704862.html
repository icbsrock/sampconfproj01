CTP : How to Perform Vulnerability Scans - TUnit Framework - Info Sec Recommendation(Automation)

Created by Lakshminarayanan Mukundagiri (TMNA), last modified by Jayasudha Yedalla (TMS) on Aug 27, 2018


Objective:   Automate the tunit framework  to perform security testing for the given app urls.


Mechanism :  RestAssured with TestNG been used to perform the automation , build thru jenkins .


How to Setup


  •  Tunit Framework backend service provided by infosec team is been deployed on CTP preprod ( https://ctpsecurity-tunit.preprod.ctdevops.com) 
  •  Automation framework developed using Rest Assured and Test NG been commited to bit bucket .

                https://bitbucket.sdlc.toyota.com/scm/ctp/ctp-security-testautomation.git

  •   Jenkins Job  is configured to execute the Security Test Automation.
            1. https://jenkins.preprod.ctdevops.com/job/ctp-security/job/ctp-securitytest-automation/
            2. Jenkins Job user unputs
                       testURL:
                       testPort:
                       product:
                       env:
                       baseURL : This url doesnot change unless the deployed backend service url is changed.


  • XL Release Task has been added to services to perform the security tests


  • On Successful XL Release run for security Test the status can be checked in releases


Sample Result File:

  All the run results are published using HTML Reports and uploaded to S3 Bucket.
