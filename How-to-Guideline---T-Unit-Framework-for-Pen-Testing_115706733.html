CTP : How to Guideline - T Unit Framework for Pen Testing

Created by Lakshminarayanan Mukundagiri (TMNA), last modified by Jayasudha Yedalla (TMS) on Aug 27, 2018


Objective :   Perform Security testing for CTP Registration apps.


Mechanism: Tunit Framework provided for Toyota Information Security Team .


How to Setup:


  •        Extract the zip file provided by InfoSec Team.
  •        Prerequisites
                    1. nix computer (has been tested on MacOS X and Linux) ,Might work on Windows.
                    2. git
                    3. Python 2 w/ pip installed.
                   4. npm installed
                   5. gauntlt
                   6. Just gem install gauntlt

  •        To deploy the service on the local execute the below command 
       
                     1.   docker-compose up
                     2.   docker ps  should list the below containers up and running 
                                 a) tunit_flask  
                                 b) tunit_node
     

  •   Launching the tool to perform  security testing on any given Test URL's
                     1. http://hostname:80
                     2. Provide the test url , test port and click execute.
                     3. OnSuccessful run the results are displayed on UI.
