CTP : Public / Private EndPoints

Created by Rock Mutchler (TMS), last modified on Oct 24, 2018

It is time to identify and close down the endpoints which do not require public consumption.  If a service is to only be requested by another internal service in the platform.

This is not a DNS issue it's a routing and target group resolution path now.  When swagger file generation is automated this will be handled further out at the edge with the gateway, and can by endpoint, DNS, type of headers, etc...
