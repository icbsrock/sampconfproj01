Note to Self


Note to Self is a very simple check out activity. It focuses on the self, without any commitments to the group.  Therefore it is a good option after group conversations without clear group action items.


Running the activity


  1. Ask the participants to grab a pen and a post it
  2. Give them one minute to reflect upon the discussion that have just finished

 Request each person to write a note to self about something he/she wants to have on his/her laptop as a reminder for the next Sprint (s)
